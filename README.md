# Number Guessing game
This is a simple number guessing game built using HTML, CSS, and JavaScript. The objective of the game is to guess a randomly generated number within a specified range.

## Features:
- Random number generation within a specified range.
- User input for guessing the number.
- Feedback on whether the guess is too high, too low, or correct.
- A counter to track the number of guesses made.

## Technologies Used:
- HTML
- CSS
- JavaScrip
